Instruções Para Instalação do Script do Raspberry no Windows

  1. baixar Pibakery
  2. Formatar o cartão de memória no formato NTF
  3. Montar os script da forma desejada no pibakery
  4. Baixar iso do raspberry neste link Ou usar a isso que já vem padrão no pibakery
  5. Caso tenha escolhido outra isso além da padrão disponibilizada, terá que anexar a imagem no diretório do pibakery no seguinte endereço C:\Program Files (x86)\PiBakery\resources\app\os E renomear a iso dependendo da versão full/lite.
